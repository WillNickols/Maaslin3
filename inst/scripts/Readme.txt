The files in this folder are example input files to run the demo tutorial in the vignette.

HMP2_taxonomy.tsv is a tab-demilited file with species as columns and samples as rows. This file was generated from the data available at https://ibdmdb.org/ . It is a subset of the taxonomy file so it just includes the species abundances for all samples.

HMP2_metadata.tsv is a tab-delimited file with samples as rows and metadata as columns. This file was generated from the data available at https://ibdmdb.org/ . It is a subset of the metadata file so that it just includes some of the fields.
